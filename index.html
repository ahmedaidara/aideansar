FAQ - ANSAR ALMOUYASSAR

Cette FAQ répond aux questions les plus fréquentes sur l'utilisation de l'application ANSAR ALMOUYASSAR. Si vous ne trouvez pas de réponse à votre question, contactez l'administrateur via la section "Contact" de l'application.

Catégories

Général

Qu'est-ce que l'application ANSAR ALMOUYASSAR ?

L'application ANSAR ALMOUYASSAR est une plateforme web qui facilite la gestion de l'association. Elle permet :

  • Aux membres de consulter leurs informations
  • Aux administrateurs de gérer les données
  • Aux responsables (trésorier, secrétaire, président) d'effectuer des tâches spécifiques comme la gestion des cotisations, des fichiers, ou des projets.

Comment accéder à l'application ?

  • Accédez à l'application via un navigateur web (Chrome, Firefox, Safari recommandés) sur un ordinateur ou un appareil mobile.
  • Utilisez l'URL officielle fournie par l'association.
  • Aucune installation n'est requise, mais une connexion Internet est nécessaire.

Qui peut utiliser l'application ?

Membres :

Accès à l'espace personnel avec leur code de membre et date de naissance.

Administrateurs :

Gestion des membres et événements avec un code d'accès spécifique.

Trésoriers :

Gestion des cotisations et statistiques avec un code d'accès.

Secrétaires :

Gestion des fichiers, messages motivationnels, et bibliothèque avec un code d'accès.

Président :

Accès complet à toutes les fonctionnalités avec un code d'accès.

Comment puis-je signaler un problème ou suggérer une amélioration ?

  • Utilisez la section "Contact" dans l'application pour envoyer un message à l'administrateur.
  • Vous pouvez également contacter l'association par email ou téléphone (coordonnées disponibles auprès des responsables).

Connexion et Espace Personnel

Comment me connecter à mon espace personnel ?

  1. Sur la page d'accueil, allez à la section "Espace Personnel".
  2. Entrez votre code de membre (fourni par l'association) et votre date de naissance (format JJ/MM/AAAA, par exemple 15/03/1990).
  3. Cliquez sur "Connexion".
  4. Vous accéderez à vos informations personnelles, cotisations, et événements.

Que puis-je faire dans mon espace personnel ?

Vous pouvez :

  • Consulter vos informations (nom, code, statut, email, téléphone, etc.).
  • Vérifier vos cotisations mensuelles et globales (payées ou non).
  • Effectuer des paiements via Wave ou Orange Money (si activé).
  • Voir les événements à venir et les messages motivationnels.

J'ai oublié mon code de membre, que faire ?

  1. Contactez l'administrateur ou le secrétaire via la section "Contact" ou par email.
  2. Fournissez votre nom complet et, si possible, votre date de naissance pour vérification.
  3. L'administrateur vous renverra votre code après confirmation de votre identité.

Pourquoi ma connexion échoue-t-elle ?

  • Vérifiez que votre code de membre est correct (sensible à la casse).
  • Assurez-vous que la date de naissance est au format JJ/MM/AAAA.
  • Si le problème persiste, contactez l'administrateur pour vérifier vos informations dans la base de données.

Comment changer ma date de naissance (mot de passe) ?

  1. Contactez l'administrateur via la section "Contact".
  2. Fournissez une preuve d'identité (par exemple, une copie de votre carte d'identité).
  3. L'administrateur mettra à jour votre date de naissance dans la base de données.

Cotisations

Comment vérifier mes cotisations ?

  1. Connectez-vous à votre espace personnel.
  2. Allez à l'onglet "Cotisations".
  3. Pour les cotisations mensuelles, les mois payés sont marqués par une coche.
  4. Pour les cotisations globales, le statut (payé/non payé) est affiché.

Comment payer une cotisation ?

  1. Dans l'espace personnel, sélectionnez l'option de paiement (Wave ou Orange Money).
  2. Pour Wave, cliquez sur le lien fourni pour effectuer le paiement.
  3. Pour Orange Money, suivez les instructions affichées (peut nécessiter l'application Orange Money).
  4. Une fois le paiement confirmé, le trésorier mettra à jour votre statut (délai de 24 à 48 heures).

Que se passe-t-il si je ne paie pas mes cotisations ?

  • Votre statut peut passe à "inactif", ce qui limite votre accès à certains événements ou services.
  • Contactez le trésorier via l'application pour discuter d'éventuelles options (échelonnement, report, etc.).

Pourquoi mes cotisations ne sont-elles pas à jour après paiement ?

  • La mise à jour peut prendre 24 à 48 heures après confirmation du paiement.
  • Vérifiez que vous avez utilisé le bon lien de paiement (Wave ou Orange Money).
  • Si le problème persiste, contactez le trésorier avec une preuve de paiement.

Événements

Comment voir les événements à venir ?

  • Sur la page d'accueil, la section "Événements" affiche les événements avec un compte à rebours.
  • Dans votre espace personnel, vous verrez les détails des événements auxquels vous êtes inscrit.

Comment m'inscrire à un événement ?

  • Certains événements sont ouverts à tous les membres actifs, d'autres nécessitent une inscription.
  • Contactez l'administrateur via la section "Contact" ou par email pour vous inscrire.
  • Vous recevrez une confirmation une fois votre inscription validée.

Pourquoi un événement affiche-t-il "Date à confirmer" ?

Cela indique que la date de l'événement n'est pas valide ou n'a pas été correctement enregistrée. Contactez l'administrateur pour obtenir des informations actualisées.

Projets et Métiers

Comment soumettre une demande de promotion de mon métier ?

  1. Allez à la section "Projets" et sélectionnez "Promotion".
  2. Remplissez le formulaire avec vos informations : nom, date de naissance, métier, expérience, description, adresse, contact, email, besoins.
  3. Soumettez le formulaire.
  4. Vous recevrez un email de confirmation dans les 24 à 48 heures.

Comment soumettre un projet pour financement ?

  1. Allez à la section "Projets" et sélectionnez "Financement".
  2. Remplissez le formulaire avec les détails du projet : nom, date de naissance, titre, description, budget, délai, besoins.
  3. Soumettez le formulaire.
  4. L'association vous contactera dans un délai de 3 à 5 jours.

Pourquoi n'ai-je pas reçu d'email de confirmation ?

  • Vérifiez votre dossier de spam ou courriers indésirables.
  • Assurez-vous que l'email saisi dans le formulaire est correct.
  • Si le problème persiste, contactez l'administrateur via la section "Contact".

Comment voir les métiers ou projets disponibles ?

  • Sur la page d'accueil, la section "Métiers et Projets" affiche une liste de cartes.
  • Cliquez sur "En savoir plus" pour voir les détails (nom, expérience, budget, description, etc.).
  • Les membres connectés peuvent également consulter ces informations dans leur espace personnel.

Espace Administratif

Comment accéder aux espaces réservés (admin, trésorier, secrétaire, président) ?

  1. Allez à la section "Chatbot" ou "Accès sécurisé" sur la page d'accueil.
  2. Entrez le code d'accès correspondant à votre rôle (fourni par le président ou l'administrateur).
  3. Si le code est correct, vous serez redirigé vers l'espace correspondant.

Quelles sont les responsabilités des différents rôles ?

Admin :

Ajoute, modifie ou supprime des membres et gère les événements.

Trésorier :

Gère les cotisations (mensuelles et globales) et consulte les statistiques via des graphiques.

Secrétaire :

Gère les affiches, livres, messages motivationnels, et la bibliothèque.

Président :

Accède à toutes les fonctionnalités, modifie les codes d'accès, et supervise les projets/métiers.

Comment changer les codes d'accès (président uniquement) ?

  1. Connectez-vous à l'espace président.
  2. Allez à la section "Gestion des codes".
  3. Entrez un nouveau code (minimum 6 caractères) pour le rôle souhaité (admin, trésorier, secrétaire, président).
  4. Cliquez sur "Enregistrer". Le nouveau code sera immédiatement actif.

Pourquoi mon code d'accès ne fonctionne-t-il pas ?

  • Vérifiez que le code est correct (sensible à la casse).
  • Assurez-vous d'utiliser le code correspondant à votre rôle.
  • Si le problème persiste, contactez le président pour vérifier ou réinitialiser le code.

Fichiers et Bibliothèque

Comment ajouter une affiche ou un livre (secrétaire) ?

  1. Connectez-vous à l'espace secrétaire.
  2. Allez à l'onglet "Affiches" ou "Livres".
  3. Pour une affiche :
    • Remplissez le formulaire avec le nom, la catégorie, et l'URL de l'image (jpg, png, gif).
    • Soumettez le formulaire. L'affiche apparaîtra dans la galerie de la page d'accueil.
  4. Pour un livre :
    • Remplissez le formulaire avec le titre, la catégorie, l'URL de la couverture (jpg, png, gif), et l'URL du PDF (fichier .pdf ou lien Google Drive/Dropbox).
    • Soumettez le formulaire. Le livre sera ajouté à la bibliothèque.

Comment supprimer une affiche ou un livre (secrétaire) ?

  1. Dans l'espace secrétaire, allez à l'onglet "Affiches" ou "Livres".
  2. Consultez la liste des fichiers ou livres.
  3. Cliquez sur "Supprimer" à côté de l'élément souhaité et confirmez.

Comment accéder à la bibliothèque ?

  1. Allez à la section "Bibliothèque" sur la page d'accueil.
  2. Utilisez la barre de recherche pour filtrer les livres par titre ou catégorie.
  3. Cliquez sur la couverture d'un livre pour télécharger le PDF.

Pourquoi l'URL d'un livre ou d'une affiche ne fonctionne-t-elle pas ?

  • Vérifiez que l'URL est valide (jpg, png, gif pour les images ; .pdf ou lien Google Drive/Dropbox pour les livres).
  • Si vous utilisez Google Drive ou Dropbox, assurez-vous que le lien est configuré pour un accès public.
  • Contactez le secrétaire si l'erreur persiste.

Messages Motivationnels

Comment voir les messages motivationnels ?

  • Sur la page d'accueil, le dernier message motivationnel s'affiche dans la section "Motivation".
  • Dans l'espace personnel, vous pouvez consulter l'historique des messages.

Comment envoyer un message motivationnel (secrétaire) ?

  1. Connectez-vous à l'espace secrétaire.
  2. Allez à l'onglet "Messages".
  3. Entrez le texte du message et cliquez sur "Envoyer".
  4. Le message apparaîtra immédiatement sur la page d'accueil et dans l'historique.

Comment supprimer un message motivationnel (secrétaire) ?

  1. Dans l'espace secrétaire, allez à l'onglet "Messages".
  2. Consultez l'historique des messages.
  3. Cliquez sur "Supprimer" à côté du message souhaité et confirmez.

Coran

Comment consulter les Juz du Coran ?

  1. Allez à la section "Coran" sur la page d'accueil.
  2. Les 30 Juz sont listés. Utilisez la barre de recherche pour filtrer par numéro de Juz (par exemple, "Juz 1").
  3. Cliquez sur un Juz pour voir son contenu (si disponible).

Pourquoi certains Juz ne s'affichent-ils pas ?

  • Si la liste est vide ou filtrée, vérifiez votre recherche (par exemple, supprimez les termes de recherche).
  • Contactez l'administrateur si le contenu ne se charge pas correctement.

Statistiques

Comment consulter les statistiques des cotisations ?

  1. Connectez-vous à l'espace trésorier.
  2. Allez à l'onglet "Statistiques".
  3. Choisissez le type de cotisation :
    • Mensuelle : Sélectionnez un mois et une année pour voir le nombre de membres ayant payé ou non.
    • Globale : Sélectionnez une cotisation globale (par exemple, "Cotisation annuelle") pour voir son statut.
  4. Un graphique (barres) affiche les données.

Pourquoi le graphique des statistiques ne s'affiche-t-il pas ?

  • Vérifiez que vous êtes connecté en tant que trésorier.
  • Assurez-vous que des données de cotisations existent dans la base de données.
  • Essayez de changer le type de cotisation ou les filtres (mois/année).
  • Si le problème persiste, contactez l'administrateur.

Problèmes Techniques

Pourquoi une section ne s'affiche-t-elle pas ?

  • Assurez-vous d'être connecté avec le bon rôle (membre, admin, trésorier, etc.).
  • Vérifiez que votre navigateur est à jour (Chrome, Firefox, Safari recommandés).
  • Essayez de vider le cache de votre navigateur ou d'utiliser un autre appareil.

Que signifie l'erreur "Conteneur introuvable" ?

  • Cette erreur indique qu'une partie de l'interface n'a pas été correctement chargée.
  • Rafraîchissez la page (F5 ou bouton de rafraîchissement).
  • Déconnectez-vous et reconnectez-vous.
  • Si l'erreur persiste, contactez l'administrateur.

Pourquoi certaines fonctionnalités sont-elles lentes ?

  • Une connexion Internet instable peut ralentir le chargement des données.
  • Si vous chargez de grandes quantités de données (par exemple, liste des membres ou fichiers), cela peut prendre du temps.
  • Contactez l'administrateur pour optimiser la base de données.
  • Essayez d'utiliser un appareil avec un navigateur plus rapide.

Sécurité

Mes données sont-elles sécurisées ?

  • Oui, l'application utilise Firebase pour stocker les données de manière sécurisée.
  • Les connexions des membres sont anonymisées, et les codes d'accès sont stockés dans une base protégée.
  • Ne partagez jamais votre code de membre ou vos codes d'accès avec d'autres personnes.

Que faire si je pense que mon compte a été compromis ?

  1. Contactez immédiatement l'administrateur via la section "Contact" ou par email.
  2. Demandez une réinitialisation de votre code de membre ou de votre date de naissance.
  3. Fournissez une preuve d'identité pour sécuriser le processus.

Divers

Comment activer le mode sombre ?

  • Allez à la section "Paramètres" (si disponible) ou cherchez l'icône de thème (soleil/lune) dans l'interface.
  • Activez le mode sombre. Il sera enregistré pour vos prochaines sessions.

L'application est-elle disponible sur mobile ?

  • Oui, l'application est accessible via un navigateur mobile (Chrome, Safari, etc.).
  • Aucune application native n'est nécessaire, mais assurez-vous d'avoir une connexion Internet stable.

Puis-je recevoir des notifications ?

  • Si votre navigateur prend en charge les notifications (et que vous avez autorisé les notifications pour le site), vous recevrez des alertes pour les nouveaux messages ou événements.
  • Pour activer les notifications, acceptez la demande de permission lors de votre première visite.
